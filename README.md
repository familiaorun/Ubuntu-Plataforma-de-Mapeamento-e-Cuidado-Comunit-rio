# Ubuntu-Plataforma-de-Mapeamento-e-Cuidado-Comunit-rio
Ubuntu é uma plataforma de mapeamento social e cuidado comunitário, criada para ouvir, registrar e fortalecer redes de apoio em territórios culturais, espirituais e periféricos. Através da escuta, dos dados e do afeto, transforma informação em ação coletiva.
